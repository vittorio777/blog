python数据结构v1

内置数据结构

#python

list

创建列表：t_list = [‘Google’, ‘Runoob’, 1997, 2000]
创建二维列表：t_list = [ [0 for i in range(5)] for i in range(5)]
索引：t_list[0]
截取：t_list[1:3]
更新：t_list[2] = 2001
添加：
①t_list.append() 列表末尾添加新的对象
②t_list.insert(index, obj) 将对象插入列表中的指定索引下
删除：
①t_list.pop([index=-1]) 移除列表中的一个元素（默认最后一个元素），并且返回该元素的值
②del t_list[2] 删除该索引项
③t_list.remove() 移除列表中某个值的第一个匹配项
列表的操作符：
①+ 号用于组合列表：[1, 2, 3] + [4, 5, 6]
②* 号用于重复列表：[‘Hi!’] * 4
列表函数：
①元素个数：len(t_list)
②元素最大值：max(t_list)
③元素最小值：min(t_list)
其他列表方法：t_list.index(obj) 从列表中找出某个值第一个匹配项的索引位置
①t_list.reverse() 反向列表中元素
②t_list.sort( key=None, reverse=False) 对原列表进行排序

  • 列表的数据项不需要具有相同的类型
  • t_list[::2] 即 t_list[start: end: span]
  • List也可被当做数组来使用，但由于列表的元素可是任何的对象，因此列表保存的是对象的指针

dict

创建字典：t_dict = { ‘abc’: 123, 98.6: 37 }
索引：dict[‘abc’]
更新：dict[98.6] = 8
添加：dict[‘School’] = “68”abc
删除键：del dict[‘abc’]
删除字典：del dict
清空字典：dict.clear()

  • 键必须是唯一的，但值则不必。
  • 值可以取任何数据类型，但键必须是不可变的，如字符串，数字或元组。

list、ndarray、dataframe、series的互换

总结

  • list 是列表,可以通过索引查找数值，但是不能对整个列表进行数值运算
  • array是数组,可以通过索引值查找值，能对整个数组进行数值运算
  • Numpy matrix必须是2维的,但是 numpy array(ndarray)可以是多维的. Matrix是Array的一个小的分支，包含于Array，所以matrix 拥有array的所有特性。运算符有所不同。
  • array可以和series一起做运算
